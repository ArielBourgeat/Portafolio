Ariel Bourgeat — Portfolio Website

Personal portfolio website showcasing selected projects in 3D, design, and interactive media.
The site focuses on a minimalist aesthetic, smooth interactions, and experimental UI elements to create an engaging browsing experience.

This portfolio was built to serve both as a professional presentation of work and as a creative playground for front-end experimentation.

Overview

  This website presents a curated archive of projects including:

    3D modeling

    Visual design

    Interactive media

    Experimental digital work

The interface is intentionally minimal, allowing the visual work to remain the central focus while still incorporating subtle interaction design and motion.

The website is composed of several sections:

    Home – Landing page introducing the portfolio

    About – Personal background and contact section

    Projects – Interactive archive of selected works

    Project Pages – Detailed view of individual projects

Design Philosophy

The visual identity of the site is based on three main ideas:

Minimalism

  The layout avoids unnecessary elements and focuses on typography, spacing, and imagery.

Interaction as Experience

  Subtle interactions such as hover states, smooth transitions, and animated components create a more engaging browsing experience.

Playfulness

  The portfolio includes small interactive elements (such as mini-games and animated sections) to make the experience feel more personal and memorable.

Key Features
Custom Cursor

  A custom circular cursor replaces the default browser pointer to provide a more immersive interface and visual feedback when interacting with elements.

Interactive Project Carousel

  The Project Archive page features a vertical looping carousel where:

    Projects scroll one at a time

    The list loops infinitely

    Hovering over a project reveals a Review button

    The project title dynamically updates with animation

    This approach replaces a traditional grid layout with a more cinematic navigation experience.

    Dark / Light Mode Toggle

    Users can switch between dark and light themes directly from the interface.

    The site was designed primarily for dark mode, but the toggle allows better accessibility and preference control.

    Live Clock

The navigation bar displays a real-time clock showing:

UIO – Hours : Minutes : Seconds (AM / PM)

This element reinforces the feeling of a live interface rather than a static page.

Smooth Animations

Animations are implemented using CSS transitions and JavaScript to create fluid interactions including:

    Hover reveals

    Carousel motion

    Title transitions

    Page section animations
